# 🌟 Aptos Crowdfunding Platform 🌟

Welcome to the **Aptos Crowdfunding Platform**, a dynamic decentralized application (dApp) powered by the **Aptos Blockchain**! 🚀 This full-stack solution brings creators and supporters together with **Soulbound NFTs (SBTs)**, milestone-based campaigns, and transparent on-chain voting. Build trust, fund dreams, and create impact—all on a secure, high-performance blockchain! 💡

---

## ✨ Key Features

- 🎨 **Soulbound Identity**: Mint unique, non-transferable **SBTs** to establish user identities.  
- 💸 **Campaign Creation**: Set clear goals and milestones for your crowdfunding projects.  
- 🔒 **Secure Donations**: Donate **Octas** (APT test coins) into a safe escrow system.  
- 🗳️ **Community Voting**: Donors approve or reject milestones, ensuring accountability.  
- 🏆 **Reputation System**: Build trust with on-chain reputation tracking for creators and donors.  
- 🌐 **Seamless Frontend**: Connect with **Petra Wallet**, sign transactions, and view live data.  

---

## 🛠️ Tech Stack

### Backend / Smart Contracts  
- ⚡ **Aptos Blockchain**: Fast, permissionless, and scalable Layer 1 blockchain.  
- 📜 **Move Language**: Secure smart contracts for identity, campaigns, escrow, and voting.  

### Frontend  
- 🎨 **Next.js + Tailwind CSS**: Modern, responsive UI with utility-first styling.  
- 🔐 **Petra Wallet SDK**: Easy wallet authentication and transaction signing.  
- 🔗 **Aptos SDK**: Smooth interaction with Move smart contracts.  

### Dev Tools  
- 🧪 **Aptos Devnet**: Test your dApp with faucet Octas.  
- 🔍 **Aptos Explorer**: Track transactions at [explorer.aptoslabs.com](https://explorer.aptoslabs.com).
